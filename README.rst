fclear
======

Clear compiled files.

.. _Python: https://www.python.org/


Installation
------------

.. sourcecode:: bash

   ~ $ python setup.py install

or can use pip

.. sourcecode:: bash

   ~ $ pip install fclear


Quick start
-----------

.. sourcecode:: bash

   ~ $ fclear python              # Remove pyc, __pycache__
   ~ $ fclear py                  # Remove pyc, __pycache__
   ~ $ fclaer vim                 # Remove swp
   ~ $ fclear vi                  # Remove swp
